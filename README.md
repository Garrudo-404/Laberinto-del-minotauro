# Laberinto-del-minotauro
En este proyecto de SED vamos a hacer una doble implementación (microcontrolador y FPGA) para desarrollar un laberinto móvil accionado por servomotores, el cual será controlado mediante un joystick por varios jugadores, y tendrá una dinámica de juego centrada en derrotar al minotauro para ganar. Para ello se van a implementar sensores IR, un display con la información de la partida, control del joystick  y una máquina de estados controlada por el microcontrolador stm32f407; por otra parte el microcontrolador estará conectado a la FPGA, que será la que controle los servomotores y maneje la información de estados que le transfiere el microcontrolador.
